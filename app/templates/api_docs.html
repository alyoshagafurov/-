API, интеграция

Ссылка на API

https://pixeloff.ru/api/v2.php?k=вашключ&u=ссылканаобъявление

При первом запросе API запускается обработка (status=searching) объявления, указанного в параметре u. При последующих запросах к API с теми же параметрами обработка уже не инициируется, а просто выводится результат действующей обработки.

Пока идёт обработка (очистка фото), будет возвращаться status=processed, когда обработка завершится — status примет значение done и станут доступны ссылки на очищенные фото.

Параметры запроса

ПараметрОписание
kКлюч доступа. Укажите ваш ключ доступа. Для работы API ключ должен иметь положительный баланс.
uURL объявления. Ссылка на объявление. Рекомендуется использовать urlencode.

Пример использования (PHP)

<?php
ini_set('max_execution_time', '300');

$url = "https://www.avito.ru/..."; // Ссылка на объявление
$pixeloff_key = "MY123KEY";        // Ваш ключ доступа
$maxQuery = 100;
$cnt = 0;

function getPixeloff() {
    global $url, $pixeloff_key;
    $str = file_get_contents(
        "https://pixeloff.ru/api/v2.php?k=$pixeloff_key&u=" . urlencode($url)
    );
    return json_decode($str);
}

$json = getPixeloff();
sleep(5);
while (!$json->status || ($json->status != 'done' && $json->status != 'error')) {
    sleep(3);
    $json = getPixeloff();
    if ($cnt > $maxQuery) {
        die('TimeOut Error!');
    } else {
        $cnt++;
    }
}

if ($json->status == 'done') {
    echo $json->title . "<br>";
    foreach ($json->photos as $val) {
        echo "<img src='$val' height='60'>";
    }
}
?>
Внимание! Приведённый выше код работает долго — он посылает запрос на обработку объявления и ожидает полного завершения. Если в объявлении много фотографий, обработка займёт время. Позаботьтесь о параметре max_execution_time. Рекомендуем разбить скрипт на две части: инициирующую обработку и принимающую результаты.
Внимание! Количество одновременных обработок ограничено! Нельзя запустить более 5 обработок одновременно — организуйте на вашей стороне очередь. Если нужно больше параллельных обработок — свяжитесь с поддержкой.

Важное предупреждение

API предназначен исключительно для автоматизации и упрощения работы с объявлениями тех авторов, которые дали вам согласие на обработку/копирование/размещение принадлежащих им текстовых и фото материалов. Мы против любых нарушений прав владельцев интеллектуальной собственности!

Получая доступ, вы безоговорочно соглашаетесь с пользовательским соглашением, в рамках которого вы не имеете права обрабатывать объявления и цифровые изображения, если вами не было получено согласие их правообладателей!
